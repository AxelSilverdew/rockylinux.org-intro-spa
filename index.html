Introduction

What is the Rocky Linux Project?

Rocky Linux is a community enterprise operating system designed to be 100% bug-for-bug compatible with Red Hat Enterprise Linux now that CentOS has shifted direction. It is currently under major intensive development by the community. It is being led by Gregory Kurtzer, founder of the CentOS project. There is currently no ETA for a release. Contributors should get in touch via the communication options offered on this site.
